### 方式1：直接上传（推荐新手）
```bash

# 2. 使用FTP客户端或主机面板上传到 public_html
# 3. 确保 .htaccess 文件被上传
```

**检查清单：**
- [ ] 所有文件已上传

## 🆘 故障排除

### 常见问题
1. **页面无法访问**
   - 检查文件路径
   - 验证.htaccess配置
   - 确认web服务器配置

2. **样式不显示**
   - 检查CSS文件路径
   - 验证MIME类型设置
   - 清除浏览器缓存

3. **移动端显示异常**
   - 检查viewport meta标签
   - 验证响应式CSS
   - 测试不同设备

4. **联系表单不工作**
   - 检查mailto链接格式
   - 验证邮件客户端配置
   - 测试不同浏览器
