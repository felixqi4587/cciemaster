# 🚀 CCIE培训网站 - 纯静态版本 v2.3.0

## 📝 项目说明

这是一个专业的CCIE认证培训网站，采用苹果公司的设计语言，提供简洁优雅的用户体验。**网站完全静态化**，使用纯HTML/CSS/JavaScript构建，无需任何服务器端配置或框架依赖。

### ✨ 核心特色

- **🎨 苹果设计风格**：采用苹果公司的设计语言，简洁优雅
- **🎯 考试成功导向**：专注实验室练习和95%通过率保证
- **🎉 智能促销系统**：12个月自动变化的美国节日主题折扣
- **🎉 动态成功案例**：每周自动更新的CCIE/CCDE通过案例轮播展示
- **📱 完美响应式**：适配所有设备，从手机到桌面
- **⚡ 极速加载**：纯静态文件，无框架依赖，加载速度极快
- **📧 便捷联系**：直接调用邮件客户端，快速响应客户询问
- **🔧 零配置部署**：上传即用，无需复杂配置

## 🚀 快速开始

### 本地预览
```bash
# 方法1：使用内置脚本（推荐）
./start.sh

# 方法2：使用Python
python3 serve.py

# 方法3：使用其他服务器
python3 -m http.server 8000
php -S localhost:8000
npx serve .
```

### 部署到主机

#### 直接上传部署（推荐）
1. 下载所有文件到本地
2. 上传整个项目到主机的 `public_html` 目录
3. 确保 `.htaccess` 文件被正确上传
4. 访问您的域名即可看到网站

#### Git部署（推荐用于更新）
```bash
# 在您的主机上克隆项目
git clone https://github.com/your-username/cciemaster.git
cd cciemaster

# 以后每次更新只需
git pull origin main
```
