**版本**: v2.7.6

## 📋 项目概述

CCIE Master是一个专业的CCIE/CCDE认证培训网站，采用苹果设计风格，提供高质量的网络技术培训服务。项目已完全移除Node.js依赖，使用纯静态HTML/CSS/JavaScript构建，可直接部署到任何静态托管服务。

## ✨ 核心特性

### 🎨 设计特色
- **苹果风格设计**: 简洁、优雅、专业的视觉体验
- **响应式布局**: 完美适配桌面、平板和移动设备
- **现代化UI**: 使用最新的CSS技术和动画效果
- **无障碍设计**: 符合WCAG标准的可访问性

### 📚 培训内容
- **CCIE企业基础设施**: 200+小时视频内容，50+实验室
- **CCIE安全**: 180+小时专业安全培训
- **CCIE数据中心**: 160+小时数据中心技术
- **CCDE设计专家**: 200+小时网络设计内容

### 💰 智能促销系统
- **动态价格显示**: 根据当前月份自动显示相应的节日促销
- **详细价格信息**: 显示原价、折扣百分比和折扣后价格
- **节日主题**: 12个月不同的节日促销主题
- **实时倒计时**: 增强紧迫感的心理学元素

### 🔧 技术架构
- **纯静态构建**: 无需服务器端处理，部署简单
- **模块化设计**: 组件化的CSS和JavaScript
- **SEO优化**: 完整的结构化数据和元标签
- **性能优化**: 懒加载、压缩和缓存策略
